FPS Microgame - Version 1.0.4
=============================

This is the FPS Microgame package. Learn to create your own FPS game with this example project.
Learn more about the project here: http://u3d.as/1E4y

This version of FPS Microgame is officially compatible with Unity 2018.4. In order to fix the
project for Unity 2019, please go to "Window" > "Package Manager" and
- select "Package Manager UI", and
- click "Remove" on the right.
